# FluidMusic (Android Kotlin + Gradle)

A modern, fluid, premium music player architecture with:

- Offline + online playback pipelines
- Room local library cache
- Retrofit-based online catalog/search APIs
- Compose-first UI with bottom-tab navigation
- Now Playing experience with progress, EQ controls, and lyrics actions
- Downloads, profile/social, and advanced settings surfaces
- Domain/data separation for scaling to production

## Feature blueprint included

- Smart recommendations, mood radio, collaborative playlists
- Crossfade, gapless, replay-gain toggles
- Data saver and offline-first mode
- Hi-res, local files, and stream-ready models

## Build

```bash
./gradlew assembleDebug
```

(Requires Android SDK + compatible JDK.)
